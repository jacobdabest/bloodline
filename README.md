# BLOODLINE

A browser game. Every player gets a random power built from genes
(element + delivery + modifier). You can take other people's powers,
stack them, fuse two into one, and give them away.

Play: https://GITHUBNAME.github.io/bloodline/

- `index.html` — the 3D world (multiplayer via peer-to-peer room codes)
- `menu.html`  — the 2D systems demo (breeding, birth codes, family trees)
- `powers.js` / `lineage.js` — the game logic on its own
- `powers.py` — the original Python prototype

## Multiplayer
One player clicks **Host a world** and gets a 6-letter code.
The other picks **Join** and types it. No server needed.
